# Textboard

匿名テキスト掲示板システム

## 特徴

- シンプルな匿名掲示板
- 複数の板に対応
- スレッド・レス機能
- データ永続化

## 使い方

1. `index.html`をブラウザで開く
2. 板を選択
3. スレッドを作成または既存スレッドに書き込み

## 技術スタック

- React
- Tailwind CSS
- localStorage (データ保存)
